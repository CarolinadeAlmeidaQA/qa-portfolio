# AI WhatsApp + CRM Testing Project

## Context
This project simulates QA validation of an AI conversational platform integrated with WhatsApp and CRM systems.

## What is being tested
- End-to-end user journeys (WhatsApp → AI → CRM)
- Appointment scheduling flows
- Data synchronization between systems
- API communication (REST)

## QA Focus
- Functional testing
- Integration testing
- E2E validation
- Business rules verification
- Data consistency checks

## Real-world relevance
This reflects testing of automated conversational workflows used in customer interaction systems.
